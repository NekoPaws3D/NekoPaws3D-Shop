Adminbereich

Dieser Bereich ist ein lokaler Produkteditor für PyCharm und GitHub Pages.

Produktverwaltung

Produkte hinzufügen, bearbeiten, löschen und anschließend eine neue products.js herunterladen.

Neues Produkt

Neue Bilder zuerst in assets/products/ kopieren und hier den relativen Pfad eintragen.

Produkte

Änderungen veröffentlichen

  1. Produkte im Adminbereich bearbeiten.
  2. products.js herunterladen anklicken.
  3. Die heruntergeladene Datei in PyCharm nach js/products.js kopieren und die vorhandene Datei ersetzen.
  4. Neue Produktbilder nach assets/products/ kopieren.
  5. Änderungen zu GitHub hochladen. GitHub Pages veröffentlicht die neue Version.

Wichtig: GitHub Pages besitzt kein Backend. Ein sicherer Adminbereich, der die öffentliche Website direkt verändert, ist ohne zusätzlichen Dienst nicht möglich. Dieser Editor erstellt deshalb die fertige Produktdatei für deinen PyCharm-/GitHub-Workflow.
